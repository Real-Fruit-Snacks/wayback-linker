Capabilities

End link rot permanently.

Wayback Linker communicates directly with the Internet Archive to ensure the sources in your notes never disappear.

Automated Archiving

Processes every external http:// and https:// link in the active Markdown note with a single command.

Vault-Wide Scan

Scans every Markdown note, confirms the scope, archives each unique URL once, and applies successful replacements across the vault.

Save Page Now Integration

Sends URLs directly to the Wayback Machine's capture endpoint for fresh snapshots.

Smart Replacement

Replaces links only when a fresh snapshot URL is successfully generated. Optionally falls back to older snapshots if needed.

Ignored Domains

Skip domains such as amazon.com from notes, vault scans, and right-click archiving, including matching subdomains.

Cancelable Runs

Stop long note or vault batches from the progress window while keeping any completed successful replacements.

Secure Credentials

Utilizes Obsidian's native secure keychain to store your Internet Archive API keys safely.

Fallback Support

Optionally falls back to the most recent existing snapshot if a fresh capture fails or times out, ensuring a working link is always preserved.

Workflow

One click. Zero dead links.

1

Trigger the command

Click the ribbon button, run Archive active note links with Wayback Machine, or choose Archive all vault links with Wayback Machine for a confirmed vault-wide pass.

2

Background Capture

The plugin securely authenticates with the Internet Archive and fires off a Save Page Now request for every unique external URL found in the selected note or vault scan.

3

Inline Replacement

As soon as a snapshot is confirmed, the original URL in your markdown file is safely replaced with the permanent web.archive.org snapshot URL.

Get Started

Install in under a minute.

Recommended

Community Plugins

  1. Open Obsidian Settings and navigate to Community Plugins.
  2. Browse and search for Wayback Linker.
  3. Install and Enable the plugin.

Manual

  1. Download main.js, manifest.json, and styles.css from the latest release.
  2. Create .obsidian/plugins/wayback-linker/ inside your vault.
  3. Place the files inside that folder and reload Obsidian.
